# AiPC
AiPC — это программа‑ассистент, которая позволяет управлять вашим компьютером с помощью нейросети Gemini.  Вы вводите запрос, программа отправляет его в Gemini и получает готовый код.  Перед выполнением вы можете просмотреть код и решить: нажать «Да» для запуска или «Нет» для отмены.   Для работы программы требуется API‑ключ Gemini
