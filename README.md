# AiPC
AiPC — это программа‑ассистент, которая позволяет управлять вашим компьютером с помощью нейросети Gemini. Вы вводите запрос, программа отправляет его в Gemini и получает готовый код. Перед выполнением вы можете просмотреть код и решить: нажать «Да» для запуска или «Нет» для отмены.

Для работы программы требуется API‑ключ Gemini, который можно получить на официальном сайте Gemini.

⚠️ Важно: всегда проверяйте полученный код перед выполнением.

❗Антивирусы могут жаловаться на инсталлер программы или саму программу, потому что программа не подписана цифровой подписью, если вы хотите убедиться что вирусов в программе нету, то вы можетеп проверить её исходный код Автор программы не несёт ответственности за возможные последствия запуска сгенерированного кода.

Получить Api ключ Gemini можно перейдя на официальный сайт https://ai.google.dev/gemini-api/docs и нажав Получить ключ, после мы нажимаем create key и называем наш ключ и копируем его

Чтобы выполнять python код нужно установить python с официального сайта, я использую 3.11, но вроде можно любой

English:
AiPC is an assistant program that lets you control your computer using the Gemini neural network. You enter a request, the program sends it to Gemini, and receives the resulting code. Before executing it, you can preview the code and decide whether to click "Yes" to run it or "No" to cancel it.

The program requires a Gemini API key, which can be obtained from the official Gemini website.

⚠️ Important: Always verify the received code before executing it.

❗Antivirus programs may report the program's installer or the program itself because it isn't digitally signed. If you want to ensure the program is free of viruses, you can check its source code. The program's author is not responsible for any consequences of running the generated code.

You can get a Gemini API key by going to the official website https://ai.google.dev/gemini-api/docs and clicking "Get Key." Then, click "Create Key," name your key, and copy it.

To run Python code, you need to install Python from the official website. I'm using 3.11, but I think you can use any.
