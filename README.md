# AiPC

AiPC — это программа‑ассистент, которая позволяет управлять вашим компьютером с помощью нейросети Gemini.  
Вы вводите запрос, программа отправляет его в Gemini и получает готовый код.  
Перед выполнением вы можете просмотреть код и решить: нажать «Да» для запуска или «Нет» для отмены.

⚠️ AiPC не является официальным продуктом Google или Gemini, а лишь использует их инструменты.

---

## Требования
- API‑ключ Gemini (можно получить на [официальном сайте](https://ai.google.dev/gemini-api/docs))  
- Python (я использую 3.11, но подойдут и другие)

## Получение API‑ключа
1. Перейдите на [официальный сайт Gemini](https://ai.google.dev/gemini-api/docs).  
2. Нажмите **Get Key**.  
3. Создайте ключ (**Create Key**), назовите его и скопируйте ключ.
